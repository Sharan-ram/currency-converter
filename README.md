# currency-converter

## Technologies

- React
- Material UI
- JWT (Authentication)
- Google Oauth (Authentication)

To run locally, do:

    git clone git@github.com:Sharan-ram/currency-converter.git
    cd currency-converter
    yarn install
    yarn start
    Open: http://localhost:3000/

App deployed to https://currency-converter-web-app.herokuapp.com/
